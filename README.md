# EV CAN Intrusion Detection System

A real-time cybersecurity monitoring dashboard for Electric Vehicle CAN networks.

## Features
- Real-time CAN message monitoring
- Attack detection visualization
- Vehicle speed analysis
- RPM monitoring
- Attack distribution charts
- Interactive Streamlit dashboard

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly

## Dataset
- can_data.csv
- attack_log.csv

## Dashboard
The dashboard detects suspicious CAN traffic and visualizes vehicle network behavior.
